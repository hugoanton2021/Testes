Lista de Descrição

A finalidade da Lista de descrições é listar um conjunto de itens e os descrever.

Demonstração:

A tag DL:

É a tag responsável iniciar e fechar a listagem.

A tag DT:

É a tag responsável por nomear o item a ser descrito.

A tag DD

É a tag responsável por descrever o item nomeado pelo tag DT.
